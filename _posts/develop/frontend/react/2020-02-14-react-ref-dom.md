# Refs

refs를 사용하기 좋은 일부 케이스

1. 포커스 제어, 텍스트 선택, 미디어 재생을 관리 할 때
2. 명령형 애니메이션을 발동시킬 때
3. 서드 파티 DOM 라이브러리를 통합할 때

\* 선언적으로 할 수 있는 작업에는 refs 사용을 피해야한다.
