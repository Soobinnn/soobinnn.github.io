# Chocolatey 란?
윈도우 용 패키지 관리자

리눅스 레드햇 계열은 yum이라는 패키지 관리자가 있고, 데비안 계열은 apt라는 패키지 관리를 choco라는 윈도우즈용 패키지 관리자를 탄생시킴.

[Chocolatey link](https://chocolatey.org/)

choco를 설치한 뒤에는 명령어 창으로 설치 할 수 있다.
```
choco install 패키지명
```
