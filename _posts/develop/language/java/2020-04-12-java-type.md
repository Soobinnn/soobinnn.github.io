# Int vs Integer

- Int

자료형

산술 연산 가능

 null 로 초기화 할 수 없음

- Integer

Wrapper 클래스

unboxing하지 않으면 산술 연산이 불가능 하지만, null값을 처리할 수 있다.

Boxing : 자료형 -> Wrapper 클래스

Unboxing : Wrpper 클래스 -> 자료형
