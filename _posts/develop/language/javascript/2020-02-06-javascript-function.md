## concat

두개의 문자열을 하나의 문자열로 만들어주는 역할을 하는 함수.

입력값을 문자열 대신 배열을 사용하면 두개의 배열을 하나의 배열로 만들어주는 역할도 함.

```
{String}.concat({String});
{Array}.concat({Array});

```

**\* push, concat 차이점**

> push()는 배열 끝에 요소를 추가하고 배열의 새길이를 반환한다. concat() 메소드는 배열을 병합하는데 사용된다. 기존 배열을 변경하지 않고 새 배열을 반환한다.
